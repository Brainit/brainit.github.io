#### <span id="item1">Windows top介绍</span>
> * windows top : 窗口置顶工具
> * 轻量: 331kb
> * 绿色: 无需安装
> * 适用场景：需要在某个窗口操作，而需要用到另一个窗口的内容

#### <span id="item2">获取windows top</span>
> * 下载链接: [top.exe](/download/top.exe) 
> * （提示Not Found请联系[brainit](mailto:brainit@163.com)）

#### <span id="item3">使用</span>
> * 打开top.exe
> * 默认快捷键: F7
> * 选择某个需要置顶的窗口
> * 按F7置顶当前窗口
